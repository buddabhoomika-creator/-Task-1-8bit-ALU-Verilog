Task 1: 8-bit ALU Design and Functional Simulation

Overview

This project implements an 8-bit Arithmetic Logic Unit (ALU) using Verilog HDL. The ALU performs arithmetic and logical operations based on the select input and is verified through functional simulation.

Operations Implemented

- Addition (ADD)
- Subtraction (SUB)
- AND
- OR
- XOR
- NOT
- Shift Left
- Shift Right

Tools Used

- Verilog HDL
- EDA Playground
- EPWave

Files Included

- alu_8bit.v (ALU Design)
- tb_alu_8bit.v (Testbench)
- Waveform Screenshot
- README.md

Simulation

The design was simulated using EDA Playground, and the output was verified using EPWave waveform analysis.

Outcome

Successfully designed, simulated, and verified an 8-bit ALU using Verilog HDL.
